Java的学习日记

Java概述

  Java是一个面对对象的语言，这里我们就要讨论一下面对对象和面对过程的具体细节。它们分别是什么含义。

面对对象

  我理解的面向对象是指分开来完成，将程序分为几个板块进行实现。我在网上搜寻的过程中看见有一位大神将面向对象比喻成“做包子”，将包子皮专门做一堆，将包子馅做另外一堆，最后将二者一并完成。

面对过程

  以过程为核心，将程序进行一步步执行。这也就是每做一个包子时，都会按照一套程序顺序进行，不会出现单独做包子皮或者单独做包子馅的现象

  以上的理解如果有误还请纠正我。

是否能进行比较

  浮点数是不能进行比较的，因为小数点后有很多位数，非常不准确，但是整型可以直接进行比较。在银行系统中，不要使用float 。

Java的数据类型转换

  把级别低的变量赋值给高的时候，自动进行转换

由于float比int高，把低级别的赋值给高级别。

  把高级别赋值给低级别时，要进行强制类型转换
